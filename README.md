# Big-Data-Zajecia-1

Zadanie z przedmiotu Big Data.

## Temat
Analiza danych o krajach świata z wykorzystaniem API REST Countries, biblioteki pandas, bazy SQLite oraz wizualizacji w matplotlib.

## Zakres zadania
Projekt obejmuje:
- pobranie danych z publicznego API REST Countries
- przetworzenie danych JSON do DataFrame
- zapis danych do bazy SQLite
- analizę danych przy użyciu zapytań SQL
- wykonanie wykresu słupkowego przedstawiającego łączną populację regionów świata

## Wykorzystane technologie
- Python
- pandas
- requests
- sqlite3
- matplotlib
- Jupyter Notebook

## Pliki
- `zadanie.ipynb` — główny notebook z rozwiązaniem zadania
- `pyproject.toml` — konfiguracja projektu i zależności
- `uv.lock` — zablokowane wersje pakietów
